Code Review: Multi-Agent PR Analysis That Catches What Humans Miss

What Is Claude Code Review?

Announced by Claude Code creator Boris Cherny on X, Code Review is a multi-agent system that dispatches a fleet of specialized agents to deeply analyze every pull request. It was built internally at Anthropic first — where code output per engineer is up 200% this year — because code review had become the bottleneck.

How It Works

When a review triggers, multiple agents run in parallel on Anthropic infrastructure, each examining your diff and surrounding codebase for a different class of problem:

  • Logic errors — flawed conditionals, off-by-one bugs, incorrect state transitions
  • Security vulnerabilities — injection risks, auth gaps, unsafe data handling
  • Edge cases — unhandled nulls, boundary conditions, race conditions
  • Regressions — changes that silently break existing behavior

After analysis, a verification step checks each candidate finding against actual code behavior to filter out false positives. Results are deduplicated, ranked by severity, and posted as inline comments on the exact lines where issues were found.

The Numbers

On large PRs (1,000+ lines changed), 84% receive findings, averaging 7.5 issues per review. On small PRs under 50 lines, that drops to 31% with an average of 0.5 issues. Across all PR sizes, less than 1% of findings are marked incorrect by engineers.

How to Trigger a Review

Once an admin enables Code Review and installs the GitHub App, reviews run automatically on new PRs. You can also trigger one manually on any PR by commenting:

@claude review

For CLI-based review of local changes before pushing, you can pipe a diff into Claude Code:

git diff main --name-only | claude -p "review these changed files for security issues"

Availability

Code Review is available in research preview for Team and Enterprise plans. Reviews are billed on token usage, generally averaging $15–25 per review depending on PR size and complexity. Admins can set monthly spend caps under claude.ai/admin-settings/usage.

Key Takeaway

Claude Code Review dispatches parallel specialized agents to catch logic errors, security issues, and regressions in your PRs — with a verification step that keeps false positives under 1%, making it a high-signal addition to your review workflow.
