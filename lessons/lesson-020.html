/ultraplan: Plan in the Cloud, Execute Anywhere

2026-04-20

Quick Quiz
What does /ultraplan do, and what are the three ways to trigger it?
Click to reveal answer
/ultraplan moves the planning phase to a cloud web UI where you can review, comment on, and refine the plan interactively. Trigger it with /ultraplan <prompt>, by including the word "ultraplan" in any message, or by choosing "Refine with Ultraplan" from a local plan's approval dialog.

Why Plan in the Cloud?

Local plan mode (Lesson 010) is great for straightforward tasks. But for complex multi-file refactors or architecture changes, reviewing a plan in a terminal can feel limiting. /ultraplan moves the planning phase to a full web UI running on Anthropic's Cloud Container Runtime, while your terminal stays free for other work.

As shared by @WesRoth on X, ultraplan "shifts the complex planning phase of AI-assisted software development from the local command line directly to the cloud."

Three Ways to Start an Ultraplan

1. The /ultraplan Command

Type the command followed by your task description:

/ultraplan migrate the auth service from sessions to JWTs

Claude clones your GitHub repo into a temporary cloud container, reads the codebase, and drafts a structured implementation plan in the web UI.

2. Keyword Detection

Include the word "ultraplan" anywhere in a normal prompt:

I need an ultraplan for refactoring payments to support multi-currency

Claude detects the keyword and automatically starts a cloud planning session.

3. From Local Plan Mode

When Claude finishes a local plan and shows the approval dialog, choose "No, refine with Ultraplan on Claude Code on the web" to send the draft plan to the cloud for interactive review.

The Interactive Review Experience

Once your plan is ready, you open it in the browser. The web UI gives you tools that a terminal simply can't:

  • Inline comments — highlight specific sections and leave feedback
  • Structured outline sidebar — jump between plan sections quickly
  • Real-time revision — ask Claude to rework sections and watch updates live

Your terminal shows a status indicator (drafting, needs clarification, or ready) so you always know where things stand.

Execute Where You Want

When you approve the plan, you choose where to run it:

  • Teleport back to terminal — click "Approve plan and teleport back to terminal" to implement locally with full access to your environment. Your terminal shows the plan with options: Implement here, Start new session, or Cancel.
  • Execute in the cloud — let Claude implement the plan in the same web session where it was reviewed, then pull the changes.

Requirements

  • Claude Code v2.1.101 or later
  • A GitHub-hosted repository
  • Pro or Max subscription

Key Takeaway

Use /ultraplan for complex tasks that need careful review — it gives you a full web UI to comment on and refine Claude's implementation plan before a single line of code is written.
