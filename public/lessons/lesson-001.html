Automate Your Workflow with Claude Code Hooks

2026-03-20
hooks automation new feature
What are Claude Code hooks, and how do you use them to auto-format files after every edit?
Click to reveal answer
Hooks are shell commands configured in your settings JSON that run before or after specific tool invocations. To auto-format on every edit, add a postTool hook targeting the Edit tool that runs your formatter (e.g., npx prettier --write $CLAUDE_FILE_PATH).

What Are Hooks?

Hooks are pre and post execution callbacks that fire before or after Claude Code runs a tool. They let you enforce policies, auto-format code, log tool usage, or block dangerous operations — all without manual intervention.

Hooks are configured in your Claude Code settings files: .claude/settings.json (project-level), .claude/settings.local.json (local overrides), or ~/.claude/settings.json (global).

Hook Types

  • preTool — runs before a tool executes. Use it for validation or security gates.
  • postTool — runs after a tool completes. Use it for formatting, linting, or notifications.

Configuration Example

Add this to your .claude/settings.json to auto-run ESLint after every file edit:

{
  "hooks": {
    "postTool": [
      {
        "tool": "Edit",
        "command": "npx eslint --fix $CLAUDE_FILE_PATH"
      }
    ]
  }
}

You can also target the Bash tool, Write tool, or omit the tool field to match all tools.

Practical Use Cases

  • Auto-formatting: Run Prettier or Black after every file edit
  • Audit logging: Append tool invocations to a log file for compliance
  • Security gates: Block edits to .env or credentials files with a preTool hook
  • Notifications: Send a Slack webhook when certain tools are used

Debugging

If a hook isn't behaving as expected, use the --debug hooks flag to see exactly when hooks fire and what environment variables are available:

claude --debug hooks

Hook failures will block the tool execution by default — this is intentional for security gates, but something to watch for with formatting hooks.

Key Takeaway

Hooks turn Claude Code into an automated pipeline — add a few lines to your settings JSON and every edit, command, or file write can trigger formatting, validation, or logging automatically.
